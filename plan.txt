==================== DAILY STUDY STRUCTURE ====================

1. Concept Learning        -> 30 minutes
2. Logic Building         -> 30 minutes
3. LeetCode Practice      -> 1–2 hours
4. Dry Run (Pen-Paper)    -> 20 minutes

======================================================================

==================== HOW TO SOLVE EACH PROBLEM ====================

Step 1: Dry Run First
- Take sample input
- Solve manually

Step 2: Brute Force
- Think simplest solution

Step 3: Optimize
Ask:
- Can I use HashMap?
- Can I use Prefix Sum?
- Can I use Sliding Window?

Step 4: Code
- Write clean code
- Proper naming

Step 5: Pattern Note
Write after solving:

==================== IMPORTANT PATTERNS ====================

1. Hashing:
- Fast lookup
- Store values for reuse

2. Prefix Sum:
- Subarray problems
- cumulative sums

3. Sliding Window:
- Continuous range problems
- optimize nested loops

4. Binary Search:
- Sorted arrays
- divide and conquer

======================================================================

==================== GOLDEN RULES ====================

DO:
- Dry run every problem
- Start with brute force
- Then optimize
- Revise daily

DON’T:
- Jump to solution
- Skip basics
- Do too many problems fast
